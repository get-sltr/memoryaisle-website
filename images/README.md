# Website Images Required

Add the following images to this folder:

## Required Images

1. **logo.png** - App logo (40x40px, transparent background)
2. **favicon.png** - Browser favicon (32x32px or 16x16px)
3. **app-screenshot.png** - App screenshot for phone mockup (approximately 256x554px)
4. **app-store-badge.svg** - Apple App Store download badge
5. **google-play-badge.svg** - Google Play download badge

## Where to Get App Store Badges

- **Apple App Store Badge**: https://developer.apple.com/app-store/marketing/guidelines/#section-badges
- **Google Play Badge**: https://play.google.com/intl/en_us/badges/

## Image Guidelines

- Use PNG format for logos and screenshots
- Use SVG format for badges (better scaling)
- Optimize images for web (compress without quality loss)
- Ensure transparent backgrounds where applicable
